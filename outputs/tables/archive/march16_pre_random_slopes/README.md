# March 16, 2026 — Pre-Random-Slopes Tables

**Status:** Archived 2026-05-04 per `audit_and_review_2026-05-04.md` Action 6.

These tables were generated by `analysis/final_analysis_pipeline.py` on 2026-03-16 using the random-intercepts specification. They were superseded in April–May 2026 by the random-slopes pipeline at `scripts/random_slopes_models.py`, which is now canonical for paper headline numbers.

**Why archived rather than deleted:** Reproducibility — these are the artifacts behind older versions of the manuscript and STATUS notes. If a future user references the older numbers (β=0.182, β=0.117, β=−0.056), this is where they came from.

**Files:**
- `table1_summary_stats.csv` — descriptive statistics by welfare regime (random-intercepts sample)
- `table2_main_results.csv`, `table2_main_results.json` — Models 1–8 random-intercepts results
- `tableA1_robustness.csv` — robustness checks
- `country_slopes.csv` — bivariate per-country OLS slopes (paper's "replication appendix" alternative to BLUPs)
- `jackknife_details.csv` — leave-one-country-out for Model 2 RTI × Liberal coefficient
- `summary_stats.csv` — additional summary statistics

**Canonical replacement files (in `outputs/tables/`):**
- `rs_results.csv` — Models 1–5 random slopes
- `rs_macro_controls.csv` — Model 3 with GDP/Gini macro controls (Claim 13)
- `rs_vs_ri_model3.csv` — direct random-slopes vs random-intercepts comparison
- `rs_jackknife.csv` — Model 3 interaction jackknife (random slopes)
- `per_country_slopes.csv` — current bivariate per-country slopes (replaces `country_slopes.csv`)
- `jackknife_single_country.csv`, `jackknife_two_country.csv` — bivariate jackknife
- `blups_country_slopes.csv`, `blups_jackknife_single.csv`, `blups_jackknife_two.csv` — BLUPs jackknife (paper's headline statistic)
