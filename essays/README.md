# Essays — Public Writing

Public-facing writing under Ben's name. Distinct from the academic work in `manuscripts/` (peer-reviewed targets) and the learning material in `docs/learning_econometrics/` (private/internal).

## Audience

Primary: AI labs as potential employers (Anthropic in particular), the AI-and-knowledge-work corner of the internet, professional connections on LinkedIn. Not academic readers, not PhD admissions committees.

Secondary: students and researchers who recognise themselves in the methodology-meets-AI-tooling questions these essays raise.

## Voice and craft rules

Both `voice-ben` and `humanize-academic` skills auto-trigger for anything in this directory.

- Long compound sentences, semicolons doing real work, no em-dashes.
- Distinctive transitions deployed deliberately: `Indeed,` `Furthermore,` `Through this perspective,` `Drawing on,` `Firstly/Secondly/Thirdly` inline.
- Citation-scaffolded prose where appropriate (academic register travels well to Substack readers; less well to LinkedIn skim audiences — calibrate per piece).
- Anchor paragraphs (opening, closing) typed by Ben from scratch for detector resistance.
- Working with Ben mode applies: `working_with_ben.md` is the calibration document; defer to it on collaboration patterns.

## Conventions

- One subdirectory per essay
- `draft.md` is always the current working version
- `notes.md` for ideation, fragments, Notion paste-ins
- `research_anchors.md` for piece-specific facts and quotes (the broader research base lives in `docs/learning_econometrics/compile_error_research_base.md`)
- `published.md` (when relevant) contains the final published version + venue link + post-publication notes

## How sessions interact with this directory

A clean session for essay drafting should:
1. Read `essays/README.md` (this file)
2. Read the relevant `draft.md` for the piece in progress
3. Auto-load `voice-ben` and `humanize-academic`
4. Reference `working_with_ben.md` on calibration questions
5. Mine the research base only as needed (don't load the whole thing into context)
